# odin-recipes

## About
A simple HTML project from the Foundations Course of The Odin Project showcasing various receipes.

- [Link to the Project](https://www.theodinproject.com/lessons/foundations-recipes)

## Self-reflection
I learned how to work with basic HTML elements and why I should use some of the attributes associated with them. I think one of the most important things I learned about attributes that I used on this project is to specify the width and height attributes in image tags to help the browser layout the page without causing the page to jump or flash. Another thing that I learned about attributes that I didn't use in this project is to specify rel="noopener norefferer" whenever I specify target="_blank" in anchor tags (links) for security reasons (such as preventing phishing attacks). I've also learned that I should prepend any relative paths with "./" to prevent unexpected issues from emerging. I also learned to work with git and to commit often.
